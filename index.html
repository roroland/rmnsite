  • Layout

  • Modular CSS

    I love CSS, always learning new things and keeping myself up to date.

    I work with progressive enhancement strategy when develop a web site, meaning emphasizing core web content first and then progressively add more nuanced and technically rigorous layers of presentation and features on top of the content as the end-user's browser/internet connection allow.

    This is made possible by using feature detection so older browsers still in use today will be able to access and navigate thru all your content, whilst modern browsers can go the full experience and enhance a user’s time spent on your site.

    I code css as stand-alone components that can work independently of their original building context,allowing me to create a true modular, reusable, encapsulated and scalable style architecture. I use Sass to build faster and efficient code, SMACSS or BEM philosophy approach and the latest css core native features (like custom properties) to deliver the best solutions for sites of all sizes and needs.

  • Responsive Web Design

    The "mobile first" philosophy ensure to ship a better, faster, tailor-made experience that will work from the smallest mobile screen thru a big large 4k screen or whatever comes in the future.

    As in my approach with css, progressive enhancement strategy for responsive web design emphasizes core web page content first and progressively adds more layers of presentation and features on top as the end-user’s browser/Internet connection allow, will provide the best experience possible for your users.
